# Race Roster Timer — Landing Page

A static landing page for Race Roster Timer, a desktop application that uploads CSV race result files to the Race Roster platform. Built with plain HTML and CSS — no frameworks or build tools.

## About the App

Race Roster Timer bridges your timing system's CSV files to the Race Roster results platform. Configure your file mappings, hit Start, and results upload automatically on a repeating interval for live race-day updates.

- Interval-based uploads with smart skip detection
- Secure OAuth authentication with automatic token refresh
- Built with Qt 6 for Windows (macOS and Linux planned)
